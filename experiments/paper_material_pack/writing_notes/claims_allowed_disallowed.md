# 结果声明边界

## A. 可以声称
1. 基于成交后高频数据构造的 LSI 能刻画短时压力代理。
2. MarketLSI 与重大市场状态变化具有事件对应性。
3. QVAR 显示市场压力、收益、波动、成交承接存在尾部分位联动。
4. onset 标签下，ML 相对 naive persistence 有稳定 PR-AUC 增量。
5. ALL 特征组相对 P 有稳定增量。
6. 修正事件级评价显示模型对部分压力 episode 有提前识别能力。
7. 预算约束评价可说明有限监测资源下的排序价值。

## B. 不应声称
1. LSI 是真实盘口流动性。
2. RGARCH 是核心贡献。
3. 计量-ML 融合成功。
4. QVAR 状态稳定提升 ML_ALL。
5. 横截面 C 单独显著。
6. 事件级预警可替代监管或交易决策。
7. 模型有因果解释。
